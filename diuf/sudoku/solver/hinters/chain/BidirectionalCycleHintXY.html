Bidirectional Cycle (XY-Cycle)

The cells {0} form a bidirectional cycle: there are exactly two ways of placing the values in these cells, forming two different possible configurations. Some values appear in a box, row, or col regardless of which configuration is correct. One of these two configurations must be correct, hence other occurrences of these values can be removed from these boxs, rows, and/or cols.
